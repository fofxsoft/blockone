Get Brand Command

In this tutorial we will be using some information from the Brands table when showing a Product detail. We will need to create an API Command that fetches this for you.

Create a Single Brand View

This is almost identical to the Products table. But we will go over it again for good measure.

Navigate to the Brands table. You can do this by entering Brands in the search field.

Click on the View icon (Arrow) on any record. Again, it doesn't matter which one.

TableView.aspx?TableName=Brands

Create the GetBrand Command

Once you have your view on the screen, copy the relitive URL. You don't need the https://test.encompass/aspx1/. You will also need to change the Search to Parameters.
The URL will look like this;
TableView.aspx?TableName=Brands&Parameters=F:BrandID~V:1~O:E

Navigate to the API Commands table.

Click Add API Command.

  • Set your Publisher ID.

  • Set the Command Name to GetBrand

  • Paste the TableView URL into the URL field.
    You will need to add &Format=JSON to the end of the URL.
    The URL should look like this;

TableView.aspx?TableName=Brands&Parameters=F:BrandID~V:1~O:E &Format=JSON

  • We will need to add some white-listed parameters, so your app can change which brand it needs.
    Enter BrandID in the Allowed Parameter Array field.

Click Add Record.
