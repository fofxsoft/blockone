Installing API Commands

Before any API Command can be used it must be installed to the Distributor's site. This is done for security, a Distributor must grant access to you to access their data.

Since your publisher test site is yours, we will be granting access to the commands you just created. In production the Distributor will do this.

Accessing the Install API Interface

In your Encompass test site, mavigate to Install APIs. You can enter Install APIs in the search field.

APIInterface.aspx

If you have many publisher records, you can filter the command list to your publisher name.

Click Install for these commands;

  • GetProducts
  • GetProduct
  • GetBrand
  • UpdateProduct
  • ProductInfo

When the command installs, you will see a Get Sample Link button.
