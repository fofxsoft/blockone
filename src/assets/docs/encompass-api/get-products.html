Get Products Command

The app we you are building will show a list of products. You will need an API command to fetch this information.

Create Products View

Navicate to the Products table. Upi can enter Products into the search field.

Click on the Columns button and select these fields;

  • Product ID
  • Product Name
  • Supplier
  • Brand
  • Product Type
  • Package
  • Carrier UPC

Click Create View.

TableView.aspx?TableName=Products

Create the GetProducts Command

Once you have your view on the screen, copy the relitive URL. You don't need the https://test.encompass/aspx1/. This URL will look like this;
TableView.aspx?SelectDisplayInParent=ProductID,ProductName,Supplier,BrandID,ProductTypeID,PackageID,UPC&TableName=Products&Parameters=F:aActive~V:1~O:E&

We will be creating these commands manually. This will help to understand API Commands better.

Navigate to the API Commands table.

Click Add API Command.

  • Set your Publisher ID.

  • Set the Command Name to GetProducts

  • Paste the TableView URL into the URL field.
    You will need to add &Format=JSON to the end of the URL.
    The URL should look like this;

TableView.aspx?SelectDisplayInParent=ProductID,ProductName,Supplier,BrandID,ProductTypeID,PackageID,UPC&TableName=Products&Parameters=F:aActive~V:1~O:E&Format=JSON

  • We will need to add some white-listed parameters, so we can paginate the Products table.
    Add SelectMaxRecords;StartRecordCount to the Allowed Parameter Array field.

Encompass uses SelectMaxRecords and StartRecordCount to control pagination. We will use this later on in the Product Editor tutorial.

Click Add Record.
