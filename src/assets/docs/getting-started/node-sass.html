Node.js, npm and Sass

You will need these frameworks and tools installed in order to build, run and test Angular apps.

Install Node.js and npm

Node.js and npm are a package. Head over to the Node.js website. Download and install.

Verify that node is working. Run node -v in the terminal.command prompt.

Install Sass

CSS on its own can be fun, but stylesheets are getting larger, more complex, and harder to maintain. This is where a preprocessor can help. Sass lets you use features that don't exist in CSS yet like variables, nesting, mixins, inheritance and other nifty goodies that make writing CSS fun again.

Verify is Sass ins installed. Run sass -v in the terminal.command prompt.

Install Sass on Linux

If you're using a distribution of Linux, you'll need to install Ruby first. You can install Ruby through your package manager. You will also need the build-essential package, available through a package manager as well.

sudo gem install sass --no-user-install

Install Sass on Mac

If you prefer the command line over an application then getting Sass set up is a fairly quick process. Sass has a Ruby dependency but if you're using a Mac, congratulations, Ruby comes pre-installed.

sudo gem install sass

Install Sass on Windows

Before you start using Sass you will need to install Ruby. The fastest way to get Ruby on your Windows computer is to use Ruby Installer. It's a single-click installer that will get everything set up for you super fast.

The installer will also install a Ruby command line powershell application that will let you use the Ruby libraries.

Sass requires the Ruby 2.4.3. Sass is not compatiable with Ruby >= 2.5. Version 2.5 will cause an error.

Open the Command Prompt and run this command;

gem install sass

Install Visual Studio Code Extentions

By default VS Code does not have intellisense for CSS, SCSS, SVG and XML files. You will need to use VS Code extentions to handle these file types.

These are available in the Visual Studio Marketplace. Install the Sass Formatter, the XML Formatter and SVG Language Support.
