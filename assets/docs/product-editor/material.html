User Interface

We will be using Angular's implimentation of the Material Design Concepts. We will be using Angular Material for this project.

Material Design is a unified system that combines theory, resources, and tools for crafting digital experiences.

Installing Angular Material

From the product-editor root folder, run this command:

npm install --save @angular/material @angular/cdk

If you still have ng serve running, you will need to kill it. You can do this by pressing ctl+c in the terminal/command prompt.

Install Material Animations

Run this command in the project root folder:

npm install --save @angular/animations

@angular/animations uses the WebAnimation API that isn't supported by all browsers yet. If you want to support Material component animations in these browsers, you'll have to include a polyfill.

Serve the Application

Go to the project directory and launch the application.

ng serve --open

The ng serve command builds the app, starts the development server, watches the source files, and rebuilds the app as you make changes to those files.

The --open flag opens a browser to http://localhost:4200/.

You should see the app running in your browser.

Import the Component Modules

Material is a large library. You don't want to include everything, this would cause unnecessarily long load times. This is why Material is designed in such a way where you need to import only the modules that you need.

Edit the src/app/app.module.ts file.

You will need to include and import the BrowserAnimationsModule and these modules.

  • MatIconModule
  • MatCardModule
  • MatInputModule
  • MatButtonModule
  • MatMenuModule
  • MatToolbarModule
  • MatSidenavModule
  • MatProgressSpinnerModule
  • MatTableModule
  • MatPaginatorModule
