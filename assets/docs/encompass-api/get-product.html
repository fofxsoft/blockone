Get Product Command

Why do we need a GetProducts and a GetProduct command? Well we are doing this to seperate our needs. GetProducts fetches multiple product records, and only the fields we want to display. This makes the call smaller. The GetProduct is used to fetch more information about a single product. This is only called when the user needs it, which gretly reduces the amount of traffic.

Create the Single Product View

Navigate to the Products table.

Click on the View icon (Arrow) on any record. It doesn't matter which product it is because your app will handle this for you.

TableView.aspx?TableName=Products

Create the GetProduct Command

Once you have your view on the screen, copy the relitive URL. You don't need the https://test.encompass/aspx1/. You will also need to change the Search to Parameters. The URL will look like this;
TableView.aspx?TableName=Products&Parameters=F:ProductID~V:1~O:E

Why do we need to change Search to Parameters? This is because the API logic can allow Parameters and not Search. Parameters are a bit different than the Search. Parameters use field identifiers. F:[Field]~ V:[Filter Value]~ O:[Filter Operator (E,NE,G,GE,L,LE)]. There are others, but for the sake of this tutorial we will only use F, V and O.

We will be creating these commands manually. This will help to understand API Commands better.

Navigate to the API Commands table.

Click Add API Command.

  • Set your Publisher ID.

  • Set the Command Name to GetProduct

  • Paste the TableView URL into the URL field.
    You will need to add &Format=JSON to the end of the URL.
    The URL should look like this;

TableView.aspx?TableName=Products&Parameters=F:ProductID~V:1~O:E &Format=JSON

  • We will need to add some white-listed parameters, so your app can change which product it needs.
    Notice that there is a ProductID field in the Parameters. You will need to allow the app to change this value.
    Enter ProductID in the Allowed Parameter Array field.

You can add any field from the table to the Parameters. This can be handy when fetching specific data from Encompass.

Click Add Record.
