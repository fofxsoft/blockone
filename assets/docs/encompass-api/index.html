Encompass API

The goal of this tutorial is to familirize yourself with the Encompass API.

By the end of this tutorial, you'll be able to login, create and edit Encompass API commands.

Before you start this tutorial, you will need to be an Encompass Publisher. Encompass needs to grant access. Once you are a publisher Encompass will give you a Publisher Name and API Token.

Overview

The Encompass API is not a typical MVC web API. It is "open ended", and allows you to create and edit your own API commands. It handles authentication at a publisher level and allows Encompass customers to choose which commands are allowed access to their data.

These commands are created using systems that support the API export. Here is a basic list of supported systems;

  • Tables (Export Only)
  • Reports (Export Only)
  • Single Table Import (Import Only)
  • EDBL Import (Import Only)

This tutorial will use Tables, Reports and Single Table Imports.

Assumptions

This tutorial will use JSON data. We assume that you are fabiliar with JSON data.
